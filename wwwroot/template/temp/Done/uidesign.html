고객사의
사용자 인터페이스(UI) 디자인을 한층 업그레이드 시켜드리겠습니다.

저희 에이치앤에스에서는 어플리케이션 개발의뢰 시스템만이 아닌 별도의 사용자 인터페이스(UI) 디자인 서비스도 운영하고 있으며, 사용자 인터페이스(UI) 디자이너가 고객사가 원하는 형태의 디자인으로 무료 또는 저렴하게 제공해 드리고 있습니다.

서비스 안내
무상 서비스
IEC Series 일괄 100EA 이상 구매시(선결제 조건)

작업 진행 : 총 5페이지 디자인 (부팅화면 + 페이지 4)

디자인 형태 : A형태 난이도, 시안 1개만 진행 됩니다.

작업 유의사항 : 무료 서비스일 경우에도 추가 1페이지 발생 및 난이도 작업상황에 따라 별도로 추가 금액이 발생됩니다.

유상 서비스
디자인 의뢰시

작업 진행 : 총 5페이지 디자인 (부팅화면 + 페이지 4) ※페이지 추가시 추가 비용이 발생할 수 있습니다.

디자인 형태 : A난이도 / B난이도 90만원 / C난이도 150만원(부가세별도)

작업 유의 사항 : 추가 1p당 추가 금액 발생 / 작업 견적에 따라 가격차 변동이 될 수 있습니다.

  • A Type _ 기본형

    단순한 형태의 디자인
    일차적인 단면적인 느낌

    화면 구성
    부팅화면 1 p + 화면 4p 기준
    추가 1p당 8만원 비용 발생

    가격안내 : 60만원 선(VAT 별도)
  • B Type _ 고급형

    다양한 효과 적용
    전체적인 컬러 효과, 버튼 디자인

    화면 구성
    부팅화면 1 p + 화면 4p 기준
    추가 1p당 11만원 비용 발생

    가격안내 : 90만원 선(VAT 별도)
  • C Type _ 프리미엄

    차별화 된 다지인 제작위주
    디테일 그림 작업, 아이콘 화

    화면 구성
    부팅화면 1 p + 화면 4p 기준
    추가 1p당 15만원 비용 발생

    가격안내 : 150만원 선(VAT 별도)
개발절차
디자인 의뢰 및 문의

사용자 인터페이스 의뢰 및 견적시에는 반드시 레이아웃 형태를 제시해주셔야만 견적 및 작업진행이 됩니다.
별도의 작업문서가 있으신 경우 작성안하셔도 되며, 문서가 없으신 분들은 아래의 문서를 다운받아 작성해서 자료를 이메일로 제시해주시기 바랍니다.
