4 IEC667/1000 Active USB 드라이버
  • 업데이트 날짜 : 2011년 04월 09일
  • 다운로드

IEC667/1000에서는 반드시 개발 PC와 연결(ActiveSync)하기 위해서는 별도로 USB 드라이버를 개발 PC에 설치 하시기 바랍니다.

[설치방법] IEC-Series 제품매뉴얼 → Part - II.개발환경 → 4-7. Mobile Device Center(Active Sync) 설치 참고
[주 의] IEC266-Series는 별도 USB 드라이버를 하실 필요가 없습니다.

7 WZC Tool
  • 업데이트 날짜 : 2013년 08월 30일
  • 다운로드

프로그램은 무선랜과 AP(Access Pont)연결을 Command방식으로 처리하는 프로그램입니다. 즉 Windows 창과 같은 GUI가 없는 형태의 설정 프로그램입니다. 무선랜을 사용하여 제품에 무선랜을 장착할 경우 연결 설정 창이(GUI)출력됩니다. 이창에서 설정하시면 됩니다.

암호화 방식 AES는 지원 관련 GUI창에서 설정 할 경우 AES방식이 지원되지만 wzctool.exe에서는 지원이 되지 않습니다.

[지원가능 제품] IEC 266/667/1000-Series
[사용 방법] IEC-Series 제품매뉴얼 → Part - V. 기타옵션제품 → 1-5.WZC Tool.exe 사용하기 참고

8 .NET Compact Framework(2.0/3.5 버전)
  • 업데이트 날짜 : 2016년 06월 17일
  • 다운로드

Windows 10, 8.1, 8에서 VisualStudio 2008 사용자가 .NET Compact Framework 관련 문제가 발생하는 경우 압축 파일을 다운로드 받고 아래와 같은 순서로 설치 바랍니다.
[STEP-1] .NET Compcat Framework 2.0 파일 설치
[STEP-2] .NET Compcat Framework 3.5 파일 설치

※.NET Compact Framework 2.0/3.5 설치해도 오류가 발생하는 경우
- .NET Compcat Framework 3.5 Power Toys 파일 설치
- 공지사항 93. Windows 10(pc의 OS) 개발환경 지원 안내 참조 하신 후 SmartX Framework 재설치 하시기 바랍니다.

9 BootLogo 이미지 변환 프로그램
  • 업데이트 날짜 : 2016년 10월 04일
  • 다운로드

SmartBootLogo를 사용하여 BootLogo 이미지를 변경할 경우 이미지의 형식이 16Bit-Bitmap565 형식으로 되어 있어야 합니다. 따라서 이미지 형식을 변환하기 위한 작업이 필요하며 이 과정에서 포토샵이나 별도의 프로그램이 필요합니다.
BootLogo 이미지 변환 프로그램은 이미지파일(BMP, JPG, PNG, GIF)을 쉽게 16Bit-Bitmap565 형식으로 변환해주는 프로그램입니다.

[주 의] IEC - Series LCD 해상도에 따라 맞는 이미지가 필요합니다.
[참 조] SmartX Programming Guide의 SmartBootLogo을 참고하시기 바랍니다.
