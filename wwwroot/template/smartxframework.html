SmartX Framework
독자적으로 개발한 새로운 개념의 Framework

. NET Compact Framework 기반의 확장된 기능을 제공하며 Embedded System 분야에서 강력한 생산성을 제공하도록 설계된 .NET 컴포넌트(라이브러리)

무료지원 솔루션 / 쉬운 설치

SmartX Framework 설치하시면 자동으로 Visual Studio 2005/2008(권장) 자동추가

지속적인 업데이트

지속적으로 컴포넌트를 관리하여 업데이트 및 새로운 기능 추가

다양한 기능 지원

H/W 제어, 사용자 인터페이스 관리 편의기능을 제공하여 쉽게 응용프로그램 개발시간 단축

Application Reference 및 예제자료

Tech Note, Connectd Device 기술 자료 및 SmartX Framework 예제 파일 제공

빠른 기술지원

고객사의 상황에 맞는 맞춤형 기술지원(원격지원) 제공

Hardware Control Class
SmartGPIO
확장 GPIO 포트 제어 컴포넌트
- GPIO Port 입/출력
- 포트각각입/출력모드제어가능
- 입력신호의 감시를 이벤트로 처리
- 비 라이트 모델의 경우에는 IEC266 : 16EA, IEC667:48EA, IEC1000 : 48EA입/출력 핀 제공
- 라이트 모델의 경우에는 IEC266 : 16EA, IEC667:16EA, IEC1000 : 16EA입/출력 핀 제공
SmartInputCounter
SmartGPIO의 입력대비 고속의 입력 처리 기능을 지원하는 컴포넌트
-SmartGPIO의 입력대비 고속의 입력 처리기능 지원(별도 MCU 처리)
-기존 IEC-Series SmartGPIO의 Input 성능 대비 약 27,000배 성능 향상
기존 SmartGPIO(Input)은 22Hz(45ms)에 비해 SmartInputCounter는 680Khz(0.00147ms)속도로 수신 가능
-최대 8채널(Ch)의 InputCounter 지원
-최대 64Bit Data Width 지원
-Rotary Encoder Counter 지원
-SmartInputCounter의 각종 기능(initialize(초기화), Run(수신시작), Pause(수신정지), Reset(카운팅값 초기화)
-CheckSum Exception 기능 지원으로 데이터 신뢰성 보장
-입력 신호의 Rising Edge에서 카운팅 값 증가
SmartADC
ADC(Analog to Digital Converter)를 제어하는 컴포넌트
- Analog 값을 Digital 값으로 입력
- IEC266-Series는 4Ch(10Bit), IEC667-Series는 4Ch(12Bit), IEC1000-Series는 6Ch(12Bit), IEC1000Lite-Series는 4Ch(12Bit), 제공
- 노이즈 성분의 신호를 저감 처리 기능 지원
SmartDAC
DAC(Digital to Analog Convert)를 제어하는 컴포넌트
-Digital 값을 Analog 값으로 출력
-최대 2Ch 제공
-12Bit(0~4095) 해상도지원
-출력 전압은 0~5V 또는 0~10V(Switch Select) 가능
SmartSerialPort
시리얼 통신을 편리하게 처리하기 위한 컴포넌트
- Event 또는 Polling방식의 수신 기능 지원
- Event 수신 시 수신 Queue 지원
- 프레임 구조의 송/수신 기능 지원
- 비 프레임 구조 송/수신 기능 지원
- Error Check 기능 지원
- 송/수신 데이터 표현 방식(Binary, ASCII, UNICODE)
- RS485 통신 지원
SmartPWM
PWM을 편리하게 제어하는 컴포넌트
- IEC-Series 제품 내부 2채널의 16Bit PWM(Pulse Width Modulation)이 내장
- Clock Divider / PWM Counter / Duty Rate의 편리한 제어
SmartModbus
Modbus 사용을 편리하게 처리하는 컴포넌트
- Modbus Master 기능
- SmartModbus는 RTU 방식만 지원
- 송/수신 제어방식이 IEC667/1000-Series는 H/W 방식, IEC667/1000Lite-Series와 IEC266-Series는 S/W방식
SmartModbusSlave
Modbus-Slave 기능을 지원하며 Event 방식 처리지원 컴포넌트
- Modbus Slave 기능
- Event 방식으로 처리하도록 설계되어 프로그램 구현이 편리
- SmartModbusSlave는 RTU 방식만 지원
SmartMemory
시스템 RAM과 외부 Storage등의 영역을 실시간 모니터링 및 설정 컴포넌트
-사용중인 시스템메모리의 상태값을 편리하게 얻음
-개발된 장치 응용프로그램의 메모리 누수(Leak)를 쉽게확인
-Flash Disk와 SD Card의 사용상태를 확인
- RAM영역의 상태를 쉽고 간편하게 볼수 있도록 모니터링 화면 제공
- 모니터링화면은 대상 응용프로그램의 상위(Top Z-Order)로 출력되며 위치 또한 편리하게 변경
- Program 메모리와 Storage 영역의 비율을 설정
- 프로그램메모리경보이벤트기능
- CPU 사용량 체크 기능
SmartIIC
IIC(Inter-Integrated Circuit) 통신을편리하게 처리하는 컴포넌트
- IEC Series에서 SmartIIC는 Master Mode로 동작 되도록 구성
- IIC(I2C)는 2개의 신호선 만으로 통신이 가능
- 여러 개의 슬레이브(Slave) 디바이스들과 1:N 통신 가능
- IIC의 SDA와 SCL은 IEC-Series의 Extension Port에서 PORTB6/SDA이며, PORTB7/SCL 사용
SmartBootLogo
부트로고를 변경할 수 있도록 지원하는 프로그램
- 부트로고 이미지의 형식이 Bitmap 565형식만 가능
- BootLogo 이미지 변환 프로그램을 사용하여 쉽게 형식을 변환할 수 있도록 프로그램 제공
SmartSound
사운드 관련 기능을 편리하게 제어해주는 컴포넌트
- MP3, Wave, WMA, MPEG1,2 등의사운드파일을간단하게플레이
- 음성안내 등 기능을 편리하게 구현 가능
SmartBattery
외부 전원 공급 차단시 시스템의 데이터 및 상태를 Backup하는 컴포넌트
- IEC Lite-Series에서만 지원
- 외부 전원 연결 시 충전 기능
- 충전 관련 보호회로 내장
- 충전 상태 및 남은 용량 확인 기능
- 외부전원 상태 확인 기능
- 충전 속도 선택 Mode 지원(Fast/Normal)
- 시스템 전원 OFF 기능 지원(Power Shutdown)
- 외부의 전원을 사용하지 않는 포터블(Portable) 시스템 구성 용도
- 외부 전원의 공급이 차단될 경우 시스템의 데이터 및 상태를 Backup하기 위한 용도
SmartVideo
별도의 옵션 제품인 SmartVideo의 사용을 위한 컴포넌트
- NTSC(National Television System Committee)의 신호를 출력하는 카메라를 장착 가능
- 입력 영상 채널수는 4채널이며 한번에 한 채널만 선택 출력
- 영상 속성 제어기능(밝기, 색조, 명암, 채도) 제공
- 동영상 저장, 정지영상 저장 기능 제공
- IEC667 비LITE 모델에서만 사용이 가능
SmartWatchDog
응용프로그램에서 처리되지 않은 예외 발생을 감지 및 처리하는 컴포넌트
- Management Code에서 발생하는 Exception을 Catch 가능
- H/W WatchDog 기능의 활성화 여부 체크기능 제공
- H/W WatchDog 기능의 활성화 / 비활성화 가능
SmartPrint
인쇄관련 기능을 편리하게 처리하는 컴포넌트
- 복잡하고 어려운 인쇄기능의 단순화
- 프린터 드라이버 별도 설치 필요없음
- 하드웨어 인터페이스 : USB Type의 프린터, PCL(Printer Command Language)1~6 / 최적PCL3
- 인쇄 용지설정, 인쇄 방향, 인쇄 해상도 설정, 인쇄 색상 모드 설정 기능 제공
- 현재 인쇄 여부 체크 가능, 각종 선/도형/이미지 등을 그리는 기능 제공
User Interface(Useful Component Class)
SmartForm
MDI 화면구성을 편리하게 구현하도록 지원하는 컴포넌트
-모델별 편리한 화면 크기 지정 기능(속성창에서 간편하게 설정)
-런타임시 불필요한 폼속성 값 변경
-다중창(MDI) 기능지원
-개발모드시 편리한 프로그램 종료기능
-장치 응용프로그램 개발모드시 작업표시줄 자동숨김 기능
(작업표시줄이 폼의 하단영역을 가리는 경우 자동숨김 처리)
-투명효과 지원되는 UI컴포넌트와연동 (SmartLabel, SmartButton, SmartCheckBox, SmartRadioButtom,SmartTrackBar, SmartListBox, SmartUpdown)
-Mouse Cursor Hide 기능
-배경이미지 설정기능
-시스템설정을 위한 특수터치 이벤트 처리 기능
-SmartInnerForm과연동 지원
SmartInnerForm
여러가지(MDI) 프로그램 성능 및 메모리의 최적화 처리를 위한 컴포넌트
- 모델 별 편리한 화면 크기 지정 기능(속성 창에서 간편하게 설정)
- 다중 창(MDI) 구현 시 자식 윈도우 기능을 함
- 모든 자식창의 컨트롤을 편리하게 접근
- 디버그 모드 시 편리한 프로그램 종료 기능
- 장치 응용프로그램 개발 모드 시 작업 표시줄 자동 숨김 기능(작업 표시줄이 폼의 하단영역을 가리는 경우 자동 숨김 처리)
- 배경 이미지 설정 기능
- 시스템 설정을 위한 특수 터치 이벤트 처리기능
- SmartForm과 연동 지원
SmartDraw
도형(이미지, 선, 호, 글자)등을 쉽게 그리도록 지원하는 컴포넌트
- Windows CE의 그래픽 처리 방식에 대한 이해없이 메소드(함수)의 호출만으로 그래픽 처리를 구현가능
- 직관적인 사용자 인터페이스 함수 (모든 그래픽 처리 함수는 SmartDraw에 구현)
- WM_PAINT 이벤트에 대해 독립적 이다. (즉 WM_PAINT 이벤트를 처리할 필요 없음)
- 더블 버퍼링 구현 필요 없음
- 화면 갱신 시 깜박임을 최소화
- 그래픽 처리성능이 Native 수준에서 이루어짐(성능향상)
- 기본적인 Chart Draw 기능이 추가
- 애니메이션(Animation) GIF 출력(Decoding) 기능
- 애니메이션 및 기타 그래픽 컨트롤 제작이 편리
- 투명 배경 이미지 출력 기능
SmartKeyboard
가상 키보드(inputPanel)의 단점을 보완한 가상 키보드 패널
- 사용자 인터페이스에 맞는 디자인적 요소 변경 기능(색상 및 배경 이미지 설정기능)
- 키보드의 크기를 자유롭게 설정 및 변경 기능
- 기본적인 숫자만을 입력하기 위한 Numerical Key Pad 기능
- 기존의 가상 키보드는 TextBox에서만 입력이 가능
(SmartKeyboard는 컨트롤속성에서 Text속성이 있는 경우 모두 적용 가능)
- 한글 입력 지원
SmartButton
기존 버튼(Button)의 기능을확장한 컴포넌트
- 단순 버튼기능이 아닌 SpecialFunction기능으로 특수 버튼 기능 설정
