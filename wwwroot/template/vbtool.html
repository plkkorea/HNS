IEC - Series 장치 응용프로그램 개발 시
Visual Studio 2008 Tool만을 사용합니다.

개발툴 Visual Studio 2008은 Windows CE 장치 응용프로그램을 지원하고 있습니다.
IEC-Series는 Windows CE 5.0/6.0을 탑재하고 있어 WindowsCE 스마트용 장치 응용프로그램 개발을 지원하는 Visual Studio 2008만을 사용하여 프로그램을 개발 하셔야 합니다.

Visual Studio 2008 구매안내(Visual Studio Volume License)

Visual Studio 2008을 사용하시려면 Visual Studio 볼륨라이선스 형태로만 구매 가능합니다.
Visual Studio 2008은 현재 판매되고 있지 않으며, 최신 버전의 Visual Studio 2017을 구매 한 후 Visual Studio 2008로 다운그레이드 하여 사용가능 합니다. 볼륨 라이선스 구매 시 Package(CD)형태로 제공하지 않으며 라이선스 번호가 발급됩니다.

볼륨라이선스 혜택볼륨라이선스 혜택 최신버전부터 하위버전까지 자유롭게 하나를 선택하여 사용하실 수 있습니다.(사용시 한 버전만 사용가능, 여러개의 버전 동시 사용불가)
볼륨라이선스(VL, Volume Licence) 소매패키지(FPP, Full Package Product)
구매형태 라이선스(계약번호) 발급됨 1개 단위로 판매
프로그램 설치 미디어와 제품매뉴얼, 제품키 동봉하여 패키지 형태로 판매
제품키 관리 라이선스 관리 사이트
(Volume Licensing service center)에서 조회가능 		제품박스 및 정품인증 스티커(COA) 보관해야 하며, 분실 시 제품키에 대한 재발급 불가
사용가능 버전 구매한 버전의 하위버전까지 자유롭게 한개의 선택하여 사용가능
(단, 사용 시 한 PC에 한 버전만 선택하여 설치가능하며 한 PC 동시에 여러버전이 설치되어 있으면 안됨) 		구매한 버전만 사용가능 (다운그레이드 안 됨)
비용 소매패키지 형태보다 가격 저렴 볼륨라이선스에 비해 가격 비쌈
다운그레이드란?

상위 버전을 구매한 후 구매 버전보다 낮은 버전을 설치하여 사용하는것을 위미하며, 1대의 PC에서 한가지 버전만 설치 가능합니다.

Visual Studio 구매버전

구매 시 구매시점 기준의 최신버전으로 볼륨라이선스로 발급되며, IEC-Series 장치 응용프로그램 개발시에는 반드시 Visual studio 2008로 다운그레이드 하여 사용하시기 바랍니다.

  • Visual Studio 2017 Professional for Volume License 다운그레이드 가능
    판매가 764,000원(VAT 별도) 구매하기
  • Visual Studio 2017 Professional with MSDN for Volume License
    판매가 1,747,000원(VAT 별도) 구매하기
  • Visual Studio 2017 Premium with MSDN for Volume License
    판매가 13,717,000원(VAT 별도) 구매하기

Visual Stuido 2017 Volume License 구매시 다운그레이드 가능한 버전

Visual Studio 2015, Visual Studio 2013, Visual Studio 2012, Visual Studio 2010, Visual Studio 2008, Visual Studio 2005

MSDN 버전

MSDN subscription 이라고 하며 MSDN subscription 별도로 구독 신청을 하거나 Visual Studio with MSDN 버전을 구입하는 경우에 해당 됩니다.
MSDN은 Microsoft Development Network의 약자로 개발자들에게 필요한 정보와 마이크로소프트에서 제공하는 개발도구들의 사용권한을 제공하여 Visual Studio, Window 7 등 마이크로소프트사의 대부분의 소프트웨어를 일정 기간 동안 합법적으로 사용할 수 있습니다. 또한 기술지원 및 여러 기술자료를 제공받을 수 있습니다. 여기에서 제공되는 소프트웨어를 개별적으로 구입을 한다면 MSDN을 통해 구입하는 것보다 몇 배의 가격에 해당되기 때문에 개발자에게는 저렴하게 개발용 소프트웨어를 구입할 수 있는 방법입니다. 그렇지만 MSDN은 일정 기간 동안만 계약되는 형태의 라이선스 방식으로 계약기간이 지나면 다시 계약 즉 비용을 지불해야 하는 부분으로 비용적인 측면에서는 부담이 될 수 있습니다.
