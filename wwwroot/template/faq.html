QIEC-Series에서 장치 응용 프로그램이 실행 중 동작을 멈출 때 체크해야 할 사항
00000

IEC1000/667-Series의 SDK를 설치 시 반드시 주의해야 하는 사항이 있으며 이를 지키지 않는 경우 설치 도중 에러가 발생합니다.
이에 해결 방법을 안내하오니 참고바랍니다.

1. IEC1000-Series 제품을 사용하거나 개발 언어가 C++ 인 경우 제품에 맞는 SDK를 반드시 설치해야 합니다.
2. IEC1000-Series인 경우 IEC1000-Series SDK(Software Development Kit)을 개발 언어에 관계없이 반드시 설치해야 합니다.
3. IEC667/266-Series 인 경우 C++즉 Visual Studio 2005/2008에서 C++로 장치응용 프로그램을 개발 할 경우 IEC667/266-Series SDK를 반드시 설치 해야 합니다.

SDK 설치 버전
IEC1000 - Series 개발언어에 무관하고 IEC1000 - Series사용시 IEC1000-Series SDK 설치
IEC667 - Series 개발언어가 C++일때 IEC667-Series SDK 설치
IEC266 - Series 개발언어가 C++일때 IEC266-Series SDK 설치

파일 다운로드 ☞ 홈페이지 www.hnsts.co.kr > [자료실] > 제품관련 > 각 SDK 설치파일 [바로가기]
IEC1000-Series의 경우 OS의 버전에 따라 설치되는 SDK가 다르며 Windows Embedded Compact 7.0은 더 이상 업데이트가 지원되지 않으므로 Windows Embedded CE 6.0 사용을 권장합니다.

!사용하시는 장비의 IEC1000-Series의 OS를 확인하여 OS에 맞는 SDK를 설치하시기 바랍니다.

[장비 OS 버전 확인 방법] 장비를 개발모드로 부팅→시작→설정→제어판→시스템→일반(탭) 버전에서 확인

OS 버전 Window Embedded CE 6.0 Window Embedded Compact 7.0
설치파일 IEC1000_6_SDK.ZIP IEC1000_7_SDK.ZIP
  • Step-1. IEC1000 SDK 설치 파일을 더블 클릭하여 설치를 진행 합니다.
  • Step-2 ~ 4. HNS홈페이지 > 자료실 > 제품관련 > IEC-Series 제품매뉴얼 > 5-6. SDK 설치하기 참고
  • Step-5. [STEP-5] 설치 유형은 꼭‘Custom’선택→NEXT클릭
  • Step-6. Documentation에서 “Entire feature will be unavailable”를 선택하여 항목을 삭제
    “Entire feature will be unavailable” 선택하여 해당 항목을 삭제 시 프로그램이 정상 설치됩니다.
참고
STEP-6의 중요 사항을 지키지 않은 경우 설치 도중 에러가 발생합니다. “Entire feature will be unavailable” 선택하지 않고 해당 항목을 설치 시 설치도중에 아래와 같은 오류가 발생합니다.
